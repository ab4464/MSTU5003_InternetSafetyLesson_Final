Project: Internet Safety Infographic

Create an Infographic

Directions: Use Piktochart.com and create an infographic that demonstrates what you have learned about a particular issue within internet safety (Ex. Password protection, Security Settings, Malware, etc...).

Infographic Requirements:

  • Introduction to the issue.
  • Include at least 3 pieces of statistical data or facts.
  • List the specific safety concerns/threats.
  • List safe behaviors/what the user can do.

Conclusion: How it will impact how you behave online? or What lessons did you learn?

**Remember an infographic is about visual representation, make sure each piece of information included on the infographic includes visuals**
